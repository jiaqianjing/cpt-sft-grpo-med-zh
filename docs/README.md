## 三个阶段是什么关系

一个基座模型（`Qwen3-1.7B-Base`），依次叠加三个阶段，每一步只加一件事，从而能**干净地归因**每一步的增益：

| 阶段 | 一句话 | 输入数据 | 优化目标 | 进步体现在 |
|------|--------|----------|----------|------------|
| **CPT** 继续预训练 | 让模型"读更多领域书" | 中文医学**原始文本**（无标注） | 预测下一个 token | **困惑度**（对领域文本更熟） |
| **SFT** 监督微调 | 让模型"学会按格式答题" | 问题→推理→答案的**成对示例** | 模仿参考回答 | **输出格式/结构**（能推理、能 `\boxed{}`） |
| **GRPO** 强化学习 | 让模型"答对得奖励" | 带**可验证答案**的选择题 | 最大化"答对"奖励 | **答对率**（在训练奖励上明显↑） |

详细逐阶段：
- [**01 · CPT**](./01_cpt.md) —— 继续预训练：喂原始医学文本，困惑度下降
- [**02 · SFT**](./02_sft.md) —— 监督微调：喂 CoT 示例，学会结构化答题
- [**03 · GRPO**](./03_grpo.md) —— 强化学习：用可验证奖励，提高答对率

## 怎么读"增益"

同一套固定评测，跑 5 个消融配置（R0–R4），做减法归因：
```
SFT 增益  = R1 − R0        (只加 SFT)
CPT 增益  = R3 − R1        (固定 SFT，再加 CPT)
GRPO 增益 = R4 − R3        (再加 GRPO)
CPT 困惑度增益 = PPL(R0) − PPL(R2)
```
三条评测轴：**知识**（对数似然选择题）、**生成**（采样 CoT 后抽 `\boxed{}` 判对错）、**困惑度**（留出医学文本）。

## 本次的诚实结论（先说在前面）

- **CPT 是唯一清晰的正增益**：困惑度 7.04 → 5.79。
- **SFT 轻微降分、GRPO 仅微增**：根因是 SFT 的 CoT 蒸馏用了**偏弱的教师**（Gemini flash-lite），
  教出的推理比 1.7B 基座自身还啰嗦、更易答错；GRPO 在**训练奖励**上确实学到了（0.40→0.625），
  但迁移到测试有限。
- 这三份文档会**如实**展示这一点——既讲清每个阶段"机制上做了什么"，也诚实标注"下游是否真的变好"。
