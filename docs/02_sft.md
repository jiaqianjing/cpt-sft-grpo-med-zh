# 阶段 02 · SFT（监督微调）

> 一句话：让模型"学会按格式答题"——给出问题时，先逐步推理，再用 `\boxed{}` 给出答案。

## 1. 喂的是什么数据

**成对的示例**：问题 →（逐步推理）→ 标准答案。共 60k 条中文医学 CoT（思维链）。

**数据出处：**

| 数据集 | 链接 | 说明 |
|--------|------|------|
| FreedomIntelligence/medical-o1-reasoning-SFT | [🤗 HF](https://huggingface.co/datasets/FreedomIntelligence/medical-o1-reasoning-SFT) | zh + zh_mix 子集，共 ~45k 条真实 CoT（思维链推理），**核心推理种子** |
| FreedomIntelligence/Huatuo26M-Lite | [🤗 HF](https://huggingface.co/datasets/FreedomIntelligence/Huatuo26M-Lite) | ~177k 医学 QA（无 CoT），采样用于扩展覆盖面 |
| michaelwzhu/ShenNong_TCM_Dataset | [🤗 HF](https://huggingface.co/datasets/michaelwzhu/ShenNong_TCM_Dataset) | ~112k 中医 QA，采样用于补充 TCM 领域广度 |

> 对无 CoT 样本额外使用 Gemini flash-lite 蒸馏生成推理链，拒绝采样保留答案正确的约 15k 条。

下面是训练集里的一条**真实样本**：

**输入（user）：**
```
问题：母乳与牛乳相比，对母乳特点的描述中错误的是（　　）。
选项：
A. 含饱和脂肪酸较多   B. 乳糖含量高   C. 含白蛋白多，酪蛋白少   D. 铁吸收率高
请逐步推理，并在最后给出 \boxed{} 形式的答案。
```

**参考回答（assistant，模型要学着模仿的目标）：**
```
首先，我们逐一分析选项：
A. 含饱和脂肪酸较多：母乳不饱和脂肪酸比例较高，饱和脂肪酸相对较低……因此该描述是错误的。
B. 乳糖含量高：母乳乳糖含量通常高于牛乳……正确。
C. 含白蛋白多，酪蛋白少：母乳乳清蛋白比例高于酪蛋白……正确。
D. 铁吸收率高：母乳铁含量虽低，但吸收率高……正确。
最终答案：\boxed{A}
```

关键：参考回答**逐个选项分析**，最后落到 `\boxed{}`。SFT 就是让模型模仿这种"作答行为"。

## 2. 训练目标（机制）

**监督学习 / 模仿**：最小化模型输出与参考回答的交叉熵。模型学的不是"新知识"，而是
**"拿到医学选择题该怎么组织回答"**——逐条评估选项、给出结论、用统一格式收尾。

## 3. 进步体现在哪：作答行为变得结构化（真实 before/after）

同一道**留出测试题**（8 岁男孩肱骨髁上骨折，桡动脉搏动弱、手指凉麻 → 问合并什么），
看 base 与 SFT 后模型的真实输出：

**base（答 D，❌ 正确答案是 B）** —— 泛泛复述病史，没有落到"血管损伤"这个关键：
```
### 逐步推理分析：
1. 病史与症状：8岁男孩……X线诊断肱骨髁上骨折……两次复位未成功……
2. 可能的诊断：肱骨髁上骨折是常见儿童肘部骨折……
3. 查体发现：肿胀、压痛……桡动脉搏动…… (泛泛而谈，未抓住肱动脉损伤)
```

**SFT 后（答 B，✅ 正确）** —— 逐条评估选项，抓住"桡动脉搏动弱+手指凉麻 = 肱动脉损伤"：
```
2. 分析查体结果：
   · 桡动脉搏动弱，手指凉、麻木：这是肱动脉损伤的典型表现……
3. 评估选项：
   · A. 主要静脉损伤：……不会直接导致手指发凉麻木和桡动脉搏动减弱……
   · B. （肱动脉损伤）…… ✅
最终答案：\boxed{B}
```

**这说明**：SFT 教会了模型**"逐选项评估 + 结构化推理 + 规范收尾"**的作答方式，这道题上正是这种
系统性分析把答案从错纠正到对。

## 4. 诚实说明（重要）

- **平均而言，本次 SFT 反而让准确率轻微下降**（生成 0.589→0.544）。上面是 SFT *帮上忙*的例子；
  但也有反例——base 有时用一句话就直接答对，而 SFT 模型**啰嗦、绕远**，偶尔把自己绕到错误选项。
- 根因：SFT 的 CoT 蒸馏用了**偏弱的教师**（Gemini flash-lite），它的推理质量还不如 1.7B 基座自身，
  相当于"让强学生模仿较弱的老师"。**换更强的教师（Claude / Gemini-2.5-Pro）重蒸馏，SFT 就有望转正。**
- 所以 SFT 阶段"机制上做了什么"是清楚的（教作答格式与结构化推理）；"下游是否更准"则取决于**数据质量**。
